# docker-node

Generic node runtime and tools for personnal use.

## usage

```bash
# run a single script
docker run --rm -v $(pwd):/app -w /app pamplemousse/node node <script.js>
```


## packages included

  * elm
  * heroku
  * http-server
  * jsdoc
  * json-server
  * js-yaml
  * prettier
  * standard
  * doctoc
  * livedown
